Recommended Settings & Extension

Add .script files to your settings

{
  "files.associations": {
    "*.script": "lua",
    "*.gui_script": "lua",
    "*.render_script": "lua",
    "*.editor_script": "lua"
  }
}

Useful Extensions

  • EmmyLua (Suggested)
  • Lua language support for Visual Studio Code
  • Lint Lua scripts with luac or luajit
  • Improved Lua syntax highlighting
  • Support go to defintion and List Document Symbols.
  • Intellisense and Linting for Lua

Json Parser

I build this snippets by using a simple Python script. It parses all json files from api docs and converts them to single snippet file. It is available in the src folder of Github repo.
